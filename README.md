# supersonicmails

Pixel-faithful clone of the [mach33media.com](https://mach33media.com/) homepage, built with Next.js (App Router), React, Tailwind CSS v4, and Framer Motion.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

See `ANALYSIS.md`, `COLORS.md`, `TYPOGRAPHY.md`, `ANIMATIONS.md`, and `TECH_STACK_ANALYSIS.md` for the reverse-engineering notes.
